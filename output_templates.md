## Template 4: Action Plan (Executable)

```markdown
## Action Plan: {idea}

### 📅 Phase 1: Research & MVP (Weeks 1-2)
- [ ] Research competitor pricing (CTO)
- [ ] Identify target customer segments (CMO)
- [ ] Build prototype architecture (CTO)
- [ ] Draft financial model (CFO)
- [ ] **Deliverable:** MVP Architecture Doc

### 📅 Phase 2: Build & Beta (Weeks 3-6)
- [ ] Develop core features (CTO)
- [ ] Create landing page (CMO)
- [ ] Set up Stripe integration (CFO)
- [ ] Write technical documentation (CTO)
- [ ] **Deliverable:** Beta Version

### 📅 Phase 3: Launch Prep (Weeks 7-10)
- [ ] Beta testing with 10 users
- [ ] Iterate based on feedback
- [ ] Prepare launch marketing
- [ ] Finalize pricing
- [ ] **Deliverable:** Production Release

### 📅 Phase 4: Growth (Weeks 11-16)
- [ ] Scale infrastructure
- [ ] Expand marketing
- [ ] Customer success
- [ ] **Deliverable:** 100+ paying customers

### Key Metrics to Track
- **Weekly Active Users (WAU)**
- **Conversion Rate**
- **Customer Acquisition Cost (CAC)**
- **Lifetime Value (LTV)**
- **Monthly Recurring Revenue (MRR)**

### Risk Mitigation
| Risk | Mitigation Strategy |
|------|-------------------|
| Low adoption | Landing page waitlist |
| Competition | Differentiated features |
| Technical debt | Regular refactoring |

### Resources Needed
- **Budget:** $X
- **Team:** CTO, CMO, 1 Dev, 1 Designer
- **Timeline:** X months to break-even
```

## Usage

These templates are automatically generated when tasks complete. The AI fills in [...] with actual content based on:
- The original idea
- Research findings
- Board member perspectives
- User feedback (if provided)

The output is always in Markdown and can be:
- Read directly
- Exported to PDF
- Pasted into Notion/Docs
- Converted to other formats
